Como este site foi feito

Este projeto nasceu de uma análise exploratória dos dados do Oscar, construída no Observable com Vega-Lite e D3.js. Os dados foram enriquecidos via API do TMDB, adicionando gêneros, orçamento, bilheteria e plataformas de streaming.
